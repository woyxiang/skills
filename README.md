# Skills

A collection of custom skills.

## Available Skills

### [freebasic](./skills/freebasic/)
Guide for programming in FreeBASIC, a free/open-source BASIC compiler syntax-compatible with QuickBASIC. Covers language fundamentals, data types, control flow, OOP, graphics, threading, and C interop with bundled documentation references.

### [lsp-builder](./skills/lsp-builder/)
Step-by-step guide for building Language Server Protocol (LSP) servers based on the LSP 3.17 specification. Covers transport layer, capabilities negotiation, text document sync, language features, and workspace features.

### [skill-creator](./skills/skill-creator)
Create new skills for AI agents

### [minecraft-nbt](./skills/minecraft-nbt)
To understand Minecraft NBT (Named Binary Tag) format.

### [scoop-manifest](./skills/scoop-manifest)
To create a scoop manifest, bucket JSON, or app manifest
