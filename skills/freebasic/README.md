# FreeBASIC Skills

A collection of skills for working with FreeBASIC, a free, open-source BASIC compiler for Windows, Linux, and DOS.

## Overview

This repository contains custom skills and evaluation benchmarks for FreeBASIC development tasks.

## Structure

- `SKILL.md` - Skill definitions and configuration
- `evals/` - Evaluation benchmarks
- `references/` - [FB-manual-1.10.1-html](https://sourceforge.net/projects/fbc/files/FreeBASIC-1.10.1/Documentation/FB-manual-1.10.1-html.zip)
